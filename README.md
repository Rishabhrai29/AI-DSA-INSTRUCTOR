🚀 AI-Powered DSA Instructor
This is a simple Flask-based web application that uses Google Gemini (Generative AI) to generate helpful responses on any topic you send.
It serves a frontend (index.html) and exposes an API endpoint /generate to interact with the AI model.

📂 Features
✅ Generate AI-powered responses based on a topic
✅ Backend built with Flask
✅ Uses Google Generative AI SDK
✅ Environment variables for secure API key management
✅ CORS enabled

🖥️ Tech Stack
Backend: Python, Flask

AI: Google Generative AI

Frontend: Static index.html
